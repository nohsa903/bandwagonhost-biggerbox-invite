## BIGGERBOX 是什么，为什么需要邀请码

搬瓦工（BandwagonHost）的 BIGGERBOX 是 2024 年 12 月推出的一款限量版套餐，配置上是更早一款 MINIBOX（入门级）的"大碗"升级版——内存、硬盘、流量全部翻倍，价格从 $29/年涨到 $37/年，机房采用美国洛杉矶 DC99，线路是三网回程 CN2 GIA。

之所以需要邀请码，是因为这类套餐属于搬瓦工的**内测性质产品**——官方用来试水机房和线路，没有正式的 SLA 在线时间保证，不公开发售，走内测通道。进入内测的方式，就是拿到一个邀请码。

值得一提的是，邀请码这个机制并没有持续太久。后续搬瓦工陆续推出 POWERBOX，就直接不需要邀请码了。随着时间推移，BIGGERBOX 本身的邀请码大多已经被用掉，官方也停止了新发放。

所以，现在搜索 BIGGERBOX 邀请码的朋友，很可能面对两种现实：

- 找到的邀请码早已经被用光；
- BIGGERBOX 本身已经没有库存，买不到了。

这不是坏消息，而是需要换一个思路。
